# Журнал реализации

## 2026-07-15

- Создан чистый проект `c6_tracker_v2`.
- Из старого проекта взяты только контакты платы и последовательность инициализации контроллера дисплея.
- Определены заголовок файла C6T2 и фиксированная запись точки с CRC32.
- Реализована дозапись текущего файла, восстановление повреждённого хвоста и накопление необработанного расстояния.
- Добавлен минимальный экран на Arduino_GFX без LVGL и дополнительных шрифтов.
- Создан новый двоичный BLE-сервис со статусом, настройками, созданием нового трека и уведомлениями о точках.
- После уточнения требований удалены внутренний планировщик PSMOO и настройка частоты координат.
- GPS оставлен в штатном режиме 1 Гц во время бодрствования; перед общим Deep Sleep отправляется `UBX-RXM-PMREQ`.
- Создан начальный веб-интерфейс Vue с подключением по Web Bluetooth, картой, импортом линейных и точечных маршрутов и кнопкой очистки маршрута.
- Реализованы список файлов и возобновляемая bulk-передача блоками по 180 байт с CRC32 и накопительным ACK.
- Исходные блоки сохраняются транзакционно и без изменения в IndexedDB; добавлены сборка и проверка полного файла C6T2 перед показом на карте.
- Добавлено автоматическое переподключение к уже выбранному трекеру после планового Deep Sleep.
- Добавлено независимое PackBits-сжатие bulk-блоков с raw-резервом для несжимаемых данных.
- Реализован экспорт отфильтрованного представления в GPX 1.1 и GeoJSON.
- Исправлена публикация веб-интерфейса: GitHub Pages переведён с Jekyll корня
  репозитория на production-сборку `web/dist` через GitHub Actions.
- Web Bluetooth адаптирован по проверенной схеме старой версии: свободный системный
  выбор устройства, `optionalServices`, восстановление разрешённого трекера,
  последовательные записи с подтверждением и устойчивое переподключение.
- Добавлена мобильная вёрстка для Oppo Find X7 Ultra с mobile viewport, safe-area,
  крупными touch-элементами и адаптивными картой, вкладками, формами и списками.
- Экспорт GPX и GeoJSON сделан явным для текущего и архивных треков; при неполной
  локальной копии экспорт автоматически выполняет возобновляемую BLE-синхронизацию.
- В настройки возвращены профили «Пешком», «Авто», «Эконом», пользовательские
  профили, автоподключение и телефонные пороги фильтрации.
- Реализована очередь настроек в `localStorage`: пакет можно сохранить во время
  сна устройства, после подключения он отправляется и подтверждается автоматически.

## Прошивка платы 2026-07-15

- Устройство: ESP32-C6FH8 revision 0.2, flash 8 МБ, USB Serial/JTAG, COM4.
- Записаны bootloader, таблица разделов, `boot_app0` и приложение размером 882 272 байта.
- Все записанные области прошли проверку hash в esptool.
- После аппаратного сброса ROM сообщил `SPI_FAST_FLASH_BOOT`, то есть запуск идёт из новой flash.
- В текущем профиле Arduino вывод `Serial` приложения не перенаправлен в USB CDC, поэтому COM4 показывает ROM-загрузчик, но не строку `C6 Tracker v2`.
